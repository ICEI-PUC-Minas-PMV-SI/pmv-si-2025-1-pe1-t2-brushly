
# 📋 Requisitos do Projeto - Brushly

## ✅ Requisitos Funcionais (RF)

| Código | Descrição                                                                 |
|--------|---------------------------------------------------------------------------|
| RF01   | O sistema deve permitir o upload de imagens no formato `.jpg`, `.png`, etc. |
| RF02   | O sistema deve exibir a imagem carregada em uma área de visualização.     |
| RF03   | O sistema deve permitir ao usuário iniciar a edição ao clicar em "Comece Agora". |
| RF04   | O sistema deve abrir um modal para selecionar a imagem.                   |
| RF05   | O sistema deve validar se o arquivo enviado é uma imagem.                 |


---

## 📌 Observações

- Os requisitos estão sujeitos a evolução conforme o desenvolvimento avança.
